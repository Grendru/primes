Подробное описание

Класс-контейнер простых чисел.

Данный класс содержит в себе контейнер простых чисел и методы работы с ним.

Конструктор(ы)

◆ Primes() [1/3]

Primes::Primes ( uint32_t  max,
char  mode 
)

Создаёт объект класса Primes.

Создает контейнер на основе входных параметров.

Аргументы
[in]maxВ зависимости от режима либо максимальное значение, до которого рассчитываются и запоминаются простые числа, либо количество простых чисел, начиная с 1, которые требуется рассчитать и запомнить.
[in]modeРежим генерации элементов контейнера ('r' - поиск простых чисел в диапозоне от 1 до max, 'q' - поиск первых max простых чисел).
Исключения
Incorrect generate mode
